All of the platforms which form HotspotBnB, communicate with each other and 3rd party services using API’s.  Here is a typical plain English conversation the Algorithm Platform will have with the Platforms

Q. Algo has received an alert from Network Status saying a users data is running low (less than 500MB remaining). Network Status, does that Hotspot have credit in its wallet ?

A. (Network Status Replies) Yes. Proceed.

Q. How much ?

A. (Network Status Replies) $20 USD.

Q. Crypto Mining API. What is the current minimum amount required to purchase more data set at?

A. (Crypto Mining Platform replies with the latest figure from 3rd parties e.g. BitRefill) $10 USD

Q. Proceed. Crypto Mining API. What are the top-up amount options available ?

A. (Crypto Mining Platform replies with the latest figure from 3rd parties) $10 USD or $20 USD
