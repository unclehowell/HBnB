Crypto Mine – HotspotBnB Docs http://beta.hotspotbnb.com/docs/ PDF's & FAQ's Fri, 22 Nov 2019 07:13:49 +0000 en-US hourly 1 https://wordpress.org/?v=5.4.1 What is the LTE Hotspot ? https://beta.hotspotbnb.com/docs/faqs/what-is-the-lte-hotspot/ Sun, 16 Sep 2018 09:29:46 +0000 http://demo.wpsmartapps.com/themes/lipi/one/?post_type=helpcenter_faq&p=177 The LTE Hotspot is the product shortname for the Solar Powered/ Battery Operated, Cellular to Wi-Fi Hotspot. The product connected via LTE to establish internet access then re-broadcasts the internet over its built-in Wi-Fi Hotspot.

The LTE Hotspot is operated with the HotspotBnB Operating System which enables locally hosted, multi-user applications to be run e.g. IPCCTV, IPTV, Smart Home Control (IoT) etc
