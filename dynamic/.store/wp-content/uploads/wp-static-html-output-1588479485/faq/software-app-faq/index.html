What is the LTE Hotspot ?

The LTE Hotspot is the product shortname for the Solar Powered/ Battery Operated, Cellular to Wi-Fi Hotspot. The product connected via LTE to establish internet access then re-broadcasts the internet over its built-in Wi-Fi Hotspot.

The LTE Hotspot is operated with the HotspotBnB Operating System which enables locally hosted, multi-user applications to be run e.g. IPCCTV, IPTV, Smart Home Control (IoT) etc
